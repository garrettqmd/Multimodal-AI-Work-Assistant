# 云帆互联科技有限公司内部规章及员工办事指南
版本号：V2.0生效日期：2026 年 1 月 1 日适用范围：全体正式员工、试用期员工、实习员工、外包驻场员工
目录
# 第一章 总则
1.1 制定目的
为规范云帆互联科技有限公司（以下简称 “公司”）内部管理流程，明确员工权利与义务，保障公司运营高效、合规、有序，适配互联网行业敏捷开发、弹性协作、数据安全、创新驱动的核心特性，统一办事标准、降低沟通成本、防范经营风险，特制定本规章及办事指南。
1.2 核心原则
合规性原则：所有管理行为符合《中华人民共和国劳动法》《中华人民共和国劳动合同法》《中华人民共和国个人信息保护法》及互联网行业监管相关法律法规；
高效性原则：简化流程、线上审批、限时办结，适配互联网公司快速迭代、敏捷响应的工作节奏；
人性化原则：实行弹性考勤、灵活办公机制，兼顾工作效率与员工职业体验；
安全性原则：严守数据安全、商业秘密、用户信息保护底线，保障互联网企业核心数字资产安全。
1.3 权责说明
本制度由人力资源部牵头制定，财务部、行政部、技术部、法务部联合修订，总经理办公会审议通过后生效；各部门负责人为制度落地执行第一责任人，全体员工必须严格遵照执行。
# 第二章 员工行为规范
2.1 职业道德规范
忠诚履职，不得利用职务之便谋取私利、收受合作方回扣、侵占公司资产，不得从事与公司主营业务存在竞争关系的兼职、创业及经营活动；
恪守诚信，如实提交工作成果、报销凭证、个人信息及各类申报材料，严禁造假、虚报、瞒报；
客户至上，对待客户、合作伙伴保持专业礼貌，严禁恶意诋毁、泄露合作方商业信息；
公平友善，杜绝职场霸凌、性别歧视、地域歧视、学历歧视，营造平等包容的协作环境。
2.2 办公行为规范
工作时间禁止长时间浏览娱乐内容、玩游戏、炒股、处理大额私人事务，禁止占用公司带宽下载非工作类大文件、运行非工作程序；
保持工位及公共区域整洁，禁止办公区吸烟、大声喧哗、堆放私人物品，会议室、茶水间使用后及时清理复原；
仪容仪表以整洁得体为标准，技术、产品、研发岗位可日常休闲着装，商务对接、对外洽谈岗位需保持专业商务形象；
践行敏捷协作，工作时段 1 小时内响应工作消息，不推诿、不拖延，跨部门协作主动同步进度、对齐目标。
2.3 红线禁止行为
严禁泄露公司源代码、产品规划、用户数据、财务数据、商业合作方案、融资计划等核心机密信息；
严禁私自安装盗版软件、恶意程序、非授权远程控制工具；
严禁造谣传谣、拉帮结派、恶意中伤同事、破坏团队稳定；
严禁利用公司设备、网络、资源开展私人业务、兼职接单、倒卖公司资产及数据。
2.4 创新与知识产权规范
鼓励员工参与技术分享、产品创新、流程优化，公司设立专项创新奖励基金；严禁抄袭竞品方案、盗用开源代码未合规标注，严格遵守开源协议及知识产权相关法律。
# 第三章 考勤管理制度
3.1 考勤模式
公司实行互联网行业弹性考勤制度，具体规则如下：
打卡方式：统一使用企业微信 / 钉钉进行定位打卡或考勤机打卡；
核心在岗时段：每日 10:00–16:00 为核心工作时段，员工必须到岗或保持在线办公状态；
标准工时：每日标准工时 8 小时，每周累计 40 小时，实行早到早走、晚到晚走的灵活排班规则。
3.2 迟到、早退、旷工界定
迟到 / 早退：10 分钟以内不计入考核；10–30 分钟，每次扣绩效分 2 分；超过 30 分钟，按事假半天核算；
旷工：无故未打卡、未履行请假手续缺勤≥4 小时，视为旷工；旷工 1 天扣除 3 倍日薪资，连续旷工 3 天或月度累计旷工 5 天，视为自动解除劳动关系；
外勤与出差：外勤需提前在系统提交外勤申请，出差经部门负责人审批后，无需日常打卡。
3.3 加班管理
项目攻坚、版本迭代等场景可安排加班，优先以调休方式补偿，不强制发放加班费；
加班需提前线上提交申请，经部门负责人审批后方可生效，每日加班时长不超过 3 小时，每月累计不超过 36 小时；
法定节假日加班，按《劳动法》规定支付 3 倍薪资，或与公司协商等额调休。
3.4 远程办公考勤
因项目需求或个人特殊情况可申请远程办公，远程办公期间需在核心工时完成打卡，保持通讯畅通，人力资源部将随机抽查在线状态，未及时响应按旷工处理。

4.2 请假流程
线上申请：通过企业微信提交请假单据，注明假期类型、时长、事由及工作交接责任人；
审批权限：1–3 天由部门主管审批；3–7 天由部门经理审批；7 天以上由人力资源部 + 总经理联合审批；
紧急请假：突发疾病、意外事故可先电话报备，24 小时内补提线上申请并上传证明材料。
4.3 销假管理
假期结束后复工当日必须在系统完成销假操作，未销假且未到岗按旷工处理；提前复工的，剩余假期可转为调休额度。
# 第五章 财务报销管理制度
5.1 报销范围
仅限员工因公司业务产生的合理、必要、真实支出，包括：市内交通费、差旅费、业务招待费、办公耗材费、项目推广费、培训费、会议费等。
5.2 报销标准
交通费：市内公共交通实报实销；紧急情况可网约车出行，单次≤50 元，月度累计≤300 元；
差旅费：一线城市住宿 350 元 / 天，二线城市 280 元 / 天，三线及以下城市 220 元 / 天；餐补 50 元 / 天，城际交通实报实销；
业务招待费：单次≤500 元，部门月度累计≤2000 元，需注明招待对象、人数及业务事由；
办公费：单笔 500 元以内可自行采购，凭票报销；超过标准需走正式采购流程。
5.3 票据要求
必须提供正规增值税发票，优先使用电子发票，发票抬头为：云帆互联科技有限公司，信息完整准确；
严禁使用收据、白条、虚假发票、个人消费类发票进行报销；
电子发票需上传至费控系统归档，系统自动查重，杜绝重复报销。
5.4 报销流程
员工在费控系统上传票据、填写报销单、关联对应项目或部门；
审批流程：部门主管审核 → 财务合规审核 → 单笔≥5000 元需总经理终审；
报销时限：费用发生后 30 天内提交申请，逾期原则上不予受理；
付款时效：审批通过后 3 个工作日内，财务统一支付至员工工资卡。
5.5 备用金借款与冲账
员工出差、项目采购可申请备用金，单笔≤5000 元；任务完成后 7 天内完成冲账核销，未及时冲账将暂停后续报销权限。
5.6 禁止报销事项
个人消费、娱乐消费、非业务相关支出、超标支出、无合法票据支出一律不予报销。
# 第六章 入职与离职管理制度
6.1 入职流程
Offer 确认：候选人收到录用通知后 3 个工作日内完成确认，并配合背景调查；
入职材料：身份证、学历学位证书、离职证明、银行卡信息、社保缴纳证明；
入职办理：人力资源部签订劳动合同、开通企业系统账号、分配工位及办公设备；
岗前培训：1 天公司制度培训 + 部门业务培训，试用期 3–6 个月（三年及以上劳动合同试用期为 6 个月）。
6.2 试用期管理
试用期员工需完成转正考核，考核维度包括工作成果、考勤纪律、团队协作、岗位职责匹配度；考核不合格者，公司可依法解除劳动合同。
6.3 离职流程
主动离职：正式员工提前 30 天、试用期员工提前 3 天提交书面离职申请；
工作交接：完整交接项目资料、客户信息、办公资产、系统权限，由交接人及部门负责人签字确认；
离职结算：交接完成后，人力资源部办理社保停缴、薪资结算，未完成交接暂缓发放剩余薪资；
被动离职：公司裁员、员工严重违纪等情形，按照《劳动法》及公司制度执行。
# 第七章 办公资产与采购管理制度
7.1 资产领用
员工入职可领用标准配置：笔记本电脑、显示器、键盘、鼠标及基础办公用品；
行政部建立电子资产台账，员工领用需签字确认，妥善保管。
7.2 资产保管与归还
办公资产仅限工作使用，禁止私自转借、改装、变卖、丢弃；
离职时须全额归还资产，人为损坏或丢失按折旧价格赔偿；
设备故障统一上报行政部维修，严禁私自拆机维修。
7.3 采购流程
小额采购（≤500 元）：部门主管审批，自行采购后凭票报销；
中等采购（500 元–5000 元）：行政部审核，统一集中采购；
大额采购（≥5000 元）：总经理审批，执行比价 / 招标流程后采购；
优先选择公司签约合作供应商，保障性价比与售后服务。
# 第八章 信息安全与保密制度
8.1 保密范围
核心机密：源代码、算法模型、产品迭代计划、用户隐私数据、财务报表、融资计划；
内部信息：组织架构、薪酬体系、绩效考核方案、内部会议纪要；
用户信息：严格遵守《个人信息保护法》，禁止泄露、倒卖、非法使用用户手机号、身份信息、行为数据等。
8.2 信息安全操作规范
系统账号密码定期更换，禁用弱密码，严禁共享公司系统账号；
远程办公必须使用公司专用 VPN，禁止在公共 WiFi 下处理核心业务数据；
机密文件传输统一使用企业微信、公司邮箱，禁止使用私人微信、私人邮箱传输；
离职员工必须立即清空设备内公司数据，注销所有系统权限。
8.3 泄密责任追究
泄露公司机密造成经济或声誉损失的，公司将追究经济赔偿；情节严重、触犯法律的，移交司法机关处理。
# 第九章 会议与沟通管理制度
9.1 会议管理规范
敏捷站会每日≤15 分钟，项目评审会、复盘会需提前预约会议室，明确议程、参会人及时长；
参会人员准时到场，手机调至静音，禁止无故缺席、迟到、早退；
会议指定专人记录纪要，24 小时内同步至团队，明确责任人与完成时限。
9.2 内部沟通规范
工作沟通优先使用企业微信，重要决策及事项保留文字记录；
跨部门协作建立固定对接人，避免多头沟通、重复沟通；
对外代表公司发言、接受采访、发布信息，须经部门负责人及总经理审批。
# 第十章 绩效与奖惩管理制度
10.1 绩效考核机制
实行 OKR+KPI 双轨考核模式，月度自评、季度评审、年度总评；考核结果直接与薪资调整、奖金发放、职位晋升挂钩。
10.2 奖励制度
基础奖励：全勤奖、季度优秀员工奖、项目攻坚奖；
创新奖励：产品创新奖、技术突破奖、流程优化奖；
特殊奖励：挽回重大经济损失、获得行业权威荣誉、重大贡献专项奖励。
10.3 处罚制度
轻微违纪：迟到早退、工作懈怠，给予口头警告并扣除绩效分；
一般违纪：虚报报销、轻度泄密、违反办公纪律，给予书面警告、停发当期奖金；
严重违纪：旷工、重大泄密、贪污受贿、职场霸凌，予以直接辞退并追究相应责任。

# 第十二章 附则
本制度未尽事宜，由人力资源部牵头补充修订，经公示后生效；
员工对制度条款有异议，可向人力资源部提交书面反馈，公司统一审议回复；
实习员工、外包驻场员工参照本制度同等执行；
本制度最终解释权归云帆互联科技有限公司所有。
